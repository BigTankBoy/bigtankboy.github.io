# GitHub Pages 업로드용 폴더

이 폴더를 그대로 GitHub 저장소 루트에 올리고 GitHub Pages를 켜면 됩니다.

## 구조
- index.html
- styles.css
- images/puppy.png

## GitHub Pages 켜는 법
1) GitHub에 새 repo 생성 후 이 폴더 내용 업로드/커밋
2) Settings → Pages
3) Source: Deploy from a branch
4) Branch: main / Folder: /(root) 선택 → Save
5) https://<아이디>.github.io/<repo>/ 로 접속
